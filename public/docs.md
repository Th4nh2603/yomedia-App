# 2. Getting Started

This section will guide you through the initial setup process to get the application running on your local machine.

## 2.1 Installation

To get the project running locally, you need to clone the repository from GitHub and install the necessary dependencies using `npm`.

**Follow these steps:**

1.  Open your terminal.
2.  Clone the repository:
    ```bash
    git clone https://github.com/example/admin-dashboard.git
    ```
3.  Navigate into the project directory:
    ```bash
    cd admin-dashboard
    ```
4.  Install dependencies:
    ```bash
    npm install
    ```

## 2.2 Configuration

The application requires a `.env` file for environment variables. Create a new file named `.env` in the root directory and add the following configuration keys:

- `API_KEY=your_api_key_here` - Your personal API key for data fetching.
- `API_BASE_URL=https://api.example.com` - The base URL for the API endpoints.

# 5. FAQ

Here are some frequently asked questions.

#### How do I reset my password?

You can reset your password from the login screen by clicking the **"Forgot Password"** link. An email will be sent to you with further instructions.

#### Where can I find my API key?

Your API key is available in your user profile settings. Navigate to `Profile` > `Settings` and look for the **"API Access"** tab.
